Methods

FCPS Data

  • Student-level Daily Attendance Records
  • Student Demographics
  • School Identifiers
  • Student level demographic covariates: sex, race, FRL, homeless/highly mobile, retained in grade indicator, EL, SWD, & grade level.
  • Student attendance - Absent, Half-Day Attendance, & Full-Day Attendance; Recoded as Full-Day Present/Not-Full Day Present
  • School Name/ID used to include school level fixed effects.

Crime Data

  • Statute Descriptions
  • Start/End Timestamps
  • Address of Crime
  • Used PostGIS (2.3.6) Tiger Geocoder to geocode the address data running PostgreSQL (9.6).
  • Privacy concerns create challenges with accessing and using data.
  • Inconsistencies in address data (e.g., same street address with different zip codes) leads to erroneous geocoding values.
  • Statute descriptions don't always translate to non-law enforcement personnel that easily.

Derived Data

  • Crime Exposure
  • Crime Proximity
  • Rolling Averages
  • Geospatial Buffer Aggregations
  • We assume students would not be exposed to crimes that take place from 0800-1600 Monday-Friday.
  • We used the geodist (Picard, 2010) package in Stata to estimate the distance from students' home addresses to the crime location.

Modeling

  • Random Forests

Modeling Challenges

  • Absences are relatively rare.
  • Multiple crimes can affect attendance decision for a given day.
  • Spacetime proximity != social proximity.
